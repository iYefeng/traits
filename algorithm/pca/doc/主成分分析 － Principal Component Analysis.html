    主成分分析Principal Component Analysis

    主成分分析，又称PCA，被誉为是“应用线性代数中最重要的成果”。它可以将数据中大量的相互关联的变量转换为一小撮被称为“主成分”的变量，从而起到对数据降维和降噪的作用。

    一个直观的例子

    假设我们有如下图所示的一个数据集，每个数据点有两个分量，表示在xx和yy上，PCA所做的是先找到一个方向，使得数据向这个方向投影的方差是最大的，比如这里PCA找到了红色箭头x′x′，然后PCA在与x′x′正交的空间中再重复寻找使方差最大的投影方向，依次循环，直到所有的方向都被用完，在这里只需要再循环一次，就是找到蓝色箭头y′y′的方向。

    这样做的意义在于，我们就知道了数据集在红色箭头的方向波动最大，这个方向代表的特征的线性组合最能区分这些数据点，而蓝色箭头的组合是噪音的可能性更大。因此，在某些情况下，我们可以考虑去掉每个数据点关于蓝色箭头方向的分量，从而更精简的表示原来的数据集，虽然这样精简会损失一部分信息量，但是PCA的作用就在于可以让我们尽可能的保留信息中最有价值的一部分。

    基的变换

    假设我们有一个包含了nn个数据点的数据集XX，X={x1,x2,⋯,xn}X={x1,x2,⋯,xn}，每个数据点都有pp维，xi=(xi1,xi2,⋯,xip)xi=(xi1,xi2,⋯,xip)。当pp很大的时候，说明数据的维度很高，因此这个数据集在被应用到各种分析算法时很可能会要承受极高的时间复杂度。因此，我们希望能够找到一个基的变换，从这个数据集中选择出最主要的部分，过滤掉噪声。

    起初我们的数据集可以被这样描述：

    X=⎡⎣⎢⎢⎢⎢⎢x11x21⋮xn1x12x22⋮xn2⋯⋯⋱⋯x1px2p⋮xnp⎤⎦⎥⎥⎥⎥⎥X=[x11x12⋯x1px21x22⋯x2p⋮⋮⋱⋮xn1xn2⋯xnp]

    这个数据集的描述是基于单位基BB的，即

    B=⎡⎣⎢⎢⎢⎢10⋮001⋮0⋯⋯⋱⋯00⋮1⎤⎦⎥⎥⎥⎥=IB=[10⋯001⋯0⋮⋮⋱⋮00⋯1]=I

    线性代数中关于线性空间的知识告诉我们，任意一组基都可以等价地被用来表示这个数据集，比如说我们有一组某人年龄和身高的数据，它既可以直接用(年龄，身高)的这种形式存放，比如：

    # 年龄（岁） 身高（厘米）
    1 14 140
    2 15 147
    3 16 156
    4 18 169
    ... ... ...

    这组数据也可以使用一种不太直接的方式来表示，比如（身高＋年龄，身高－年龄）

    # 身高（厘米）＋ 年龄（岁） 身高（厘米）－ 年龄（岁）
    1 154 136
    2 162 132
    3 172 140
    4 187 151
    ... ... ...

    虽然第二种数据的表示方法麻烦而不直观，实际中我们肯定不会选用这种表示。不过，这两种表示方法在数学上是等价的，因为每种表示方法都保留了全部的信息，它们可以相互计算出对方。线性代数中我们用基的概念来表示不同的表示方法，比如第一种表示方法就是基于单位基II的，因为每个测量到的数据点(年龄，身高)，对基应用矩阵乘法得到的就是第一种方法表示的结果。

    [年龄身高]×[1001]=[年龄身高][年龄身高]×[1001]=[年龄身高]

    而第二种表示方法，使用的是基为[11−11][1−111]。因为：

    [年龄身高]×[11－11]=[身高＋年龄身高－年龄][年龄身高]×[1－111]=[身高＋年龄身高－年龄]

    一组基从本质上来看就是一个矩阵（必须是非奇异矩阵），我们写出一组基（一个矩阵）PP，然后用数据集XX乘以这个矩阵，得到的新数据集XPXP就是XX在PP上的表示，这就是完成了一次数据集基的转换。

    XP=X×PXP=X×P

    如果我们用矩阵PP的逆矩阵作为一组基应用到XPXP上，我们可以又将XPXP转化回XX。即

    XP×P−1=X×P×P−1=XXP×P−1=X×P×P−1=X

    所有的信息都没有丢失，因此只要PP可以求逆（非奇异矩阵），那么数据集变换到基PP上的表示XPXP就和XX是等价的。

    从基的概念我们看出，同样的数据在不同的基下有无穷多种表示方式，那么对于一组给定数据XX，我们很容易想到两个问题。

      选定哪个标准来定义XX不同表示方法间的优劣； 如何计算基PP使得XPXP是最优的表示方法。

    衡量不同表示方法优劣的标准

    答案就是方差， 从一个很直观的角度来说，测量到的数据中，差异化越大的地方往往才是我们要关心的。因为数据可以由两部分组成，有用的信息和噪音，而噪音必须控制在方差很小的范围内，否则的话一个噪音比信息还要大的数据集是没有任何分析价值的。为此还有一个专门的术语，信噪比（Signal-to-Noise Ratio），又称SNR，它被定义为信息／信号的方差除以噪音的反差。

    SNR=σ2信息σ2噪音SNR=σ信息2σ噪音2

    高信噪比（SNR远大于1）说明这个数据是精准的，而低信噪比说明这个数据被噪声污染得很厉害。因此在分析数据前，一个隐含的假设是我们的数据是有价值的，即数据有较高的信噪比，在这个前提下，方差较大的部分应该是有用的信息，因此能将数据投影到方差最高的方向应该是一个好的表示。

    PCA的原理

    一组基PP可以看成是pp个列向量构成(这里pp是数据的维数)

    P=[P1P2⋯Pp]P=[P1P2⋯Pp]

    这里每个PiPi都代表一个方向，在做基的变换时，新的数据表示XPXP可以看成是XX在PiPi上的投影。

    X×P=X×[P1P2⋯Pp]=[XP1XP2⋯XPp]X×P=X×[P1P2⋯Pp]=[XP1XP2⋯XPp]

    这时候你大概已经猜到PCA的工作原理了，PCA通过逐渐构建使方差最高的投影方向来构建一组基PP，使得新的XPXP更能体现出数据的本质。

    1. X1=XX1=X，令t=1,2,⋯,pt=1,2,⋯,p；
    2. 计算单位长度向量PtPt，使得Var(XtPt)Var(XtPt)最大；
    3. 令Xt+1=Xt−DXtXt+1=Xt−DXt，其中DD是一个对角矩阵，对角元DiiDii为XtPtXtPt的第ii个元素，这个过程的意义在于一旦选中了一个方向PtPt，就将数据投影到这个PtPt的正交空间中，以保证PP矩阵是一组正交基；
    4. 令t=t+1t=t+1，循环。

    最终生成的PP矩阵就是我们要求的基。P1P1被称为第一主成分，P2P2被称为第二主成分，以此类推。我们可以选择一个截断点，比如kk，然后只用前kk主成分来降维原始数据。

    X′=X×[P1P2,⋯Pk]X′=X×[P1P2,⋯Pk]

    新得到的X′X′从原来的pp维度降低到了kk维，虽然这样做会丢失原数据集中的一些信息，但是它已经尽可能的保留了最主要的那部分信息。

    PCA的计算

    PCA的计算通常使用奇异值分解算法（Singular Value Decomposition），即SVD。SVD将一个n×pn×p的矩阵XX分解为三个矩阵U,V,ΣU,V,Σ的乘积。

    X=UΣVTX=UΣVT

    其中UU和VV分别是n×nn×n，p×pp×p的单位正交矩阵，而ΣΣ是一个n×pn×p的对角矩阵。单位正交阵说明UU和VV都是基，实际上VV在这里就是我们要找的基PP。

    XP=XVXP=XV

    Iris数据举例

    在R语言中，有princomp函数专门用于进行数据的主成分分析。这里用著名的Iris数据距离如何使用PCA进行降维。

    Iris数据有5个维度，每个数据点纪录一朵山鸢花的（花瓣长度，花瓣宽度，叶子长度，叶子宽度，种类），因为维度的关系，无法将整个数据集在二维空间中画图展示。但在R里运行如下代码分析Iris数据的前四个维度

    #R语言代码
data(iris)

model <- princomp(iris[,-5])
print(model$sdev)
print(model$loadings)

    我们可以得知，Iris数据分解到的四个主成分方向P1,P2,P3,P4P1,P2,P3,P4构成的基PP为

    P=⎡⎣⎢⎢⎢⎢0.3610.0840.8570.358−0.657−0.7300.1730.075−0.5820.5980.0760.5460.315−0.320−0.4800.754⎤⎦⎥⎥⎥⎥P=[0.361−0.657−0.5820.3150.084−0.7300.598−0.3200.8570.1730.076−0.4800.3580.0750.5460.754]

    Iris数据投影到4个主方向的的标准差分别为2.049,0.491,0.279,0.1542.049,0.491,0.279,0.154，可以看到第一个主成分保留了最多的信息，而第二三四个主成分保留的信息依次减少。因此如果要做降维的话，一般可以考虑保留前两个主成分的信息。

    现在我们用前两个主成分把Iris数据降到两维，如下图：

    可以看到3类山鸢花在这两个维度上有很好的区分度，如果只基于这两个维度来聚类或者是分类，可以预见依然能取得不输于基于四个维度的聚类和分类。
