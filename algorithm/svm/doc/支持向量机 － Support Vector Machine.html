    支持向量机Support Vector Machine

    SVM(Support Vector Machine)，中文名为支持向量机。 SVM是一种有监督的机器学习算法，它可以在多维空间中构建一个（有时候是一组）超平面。这些超平面将整个空间划分为多个部分，从而达到对空间中的数据进行分类的目的。

    SVM最早在1992年被Bernhard E. Boser , Isabelle M. Guyon , Vladimir N. Vapnik在他们的论文《 A Training Algorithm for Optimal Margin Classifiers》中提出。该算法随后由于在应用于手写数字识别上的巨大成功，迅速成为了继神经网络之后的一大研究热点。尽管现在又有了新的研究热点深度学习，但是SVM依然在机器学习领域有着重要的地位。

    由于SVM算法所涉及的范围非常广泛，本文仅仅介绍SVM算法基本的思想。

    记号

    我们使用xx来表示一个样本点，从x1,x2x1,x2一直到xnxn分别表示nn个样本点。每个样本点是一个pp维的列向量，xijxij表示第ii个样本点中第jj个特征的值，同时，每个样本点xixi还对应一个类标号yiyi，yi∈−1,1yi∈−1,1。所有类标号为1的样本点集合称为正类，所有类标号为－1的样本点集合称为负类。

    xi=⎡⎣⎢⎢⎢⎢xi1xi2…xip⎤⎦⎥⎥⎥⎥xi=[xi1xi2…xip]
    yi=1或者－1yi=1或者－1

    样例数据

    样本空间的维数可以是任意维度，但是为了画图和展示的方便，一般使用二维(p=2p=2)的例子。

    在下面的例子中，我们使用的是经典数据集Iris，Iris数据集是关于鸢尾属植物，它记录了许多株鸢尾属植物的花瓣和叶子的长宽以及它们在分类学上的地位。为了方便演示SVM，需要稍微简化一下这个数据集。

    #R语言代码 简化Iris数据集
data(iris)
levels(iris$Species)[-1] <- 'non-setosa'
iris <- iris[,c(1,2,5)]

    ＃R语言代码 ggplot2绘图
library(ggplot2)
fig <- ggplot(iris,aes(x=Sepal.Length, y=Sepal.Width)) + geom_point(aes(colour=Species))
print(fig)

    图中每个点代表一株花，横轴(X轴)坐标是每株花的花瓣长度，纵轴(Y轴)坐标是每株花的花瓣宽度。红色的点表示这株花属于山鸢尾，青色点代表这株花不属于山鸢尾。我们希望能学习到可以区分一株花是否属于山鸢尾的分界超平面(决策边界)。

    边缘距离

    在这个数据集中，两种颜色表示了两类不同的样本点。而且我们可以找到一个超平面，使得红色点在这个超平面的一侧，而青色点在另外一侧，能找到符合这种超平面的数据集被称为”线性可分的“。然而一个线性可分的数据集的这种超平面不是唯一的，比如下图三条不同颜色的线都是符合条件的超平面。

    这些超平面用在线性可分的训练数据上显然误差都是00，但是我们希望找到一个在未知数据集上也能很好分割两类的超平面，为此我们要定义一个叫边缘距离的概念。

    首先上一张图说明什么是距离

    对于一个超平面，它到每一个数据点都有一个距离，其中离超平面距离最小，看上去最近的那个样本点到超平面的距离称为这个超平面的边缘距离。

    SVM求的超平面是所有能将样本点分到两类的超平面中边缘距离最大的超平面。

    你可以理解为要在两类点中间找一个超平面，要使得这个超平面不要离任一类点太近了，尽可能的从比较中间的位置剖开这两类点。因此，这样的一个超平面到两类的点边缘距离一定是相等的，假设下图这条线是最优划分超平面，那么它到红点的边缘距离和到青点的边缘距离应该是一样的。

    接下来我们先推导样本点到超平面的距离公式。

    在pp维的样本空间中，一个超平面可以用一个pp维列向量ww加上一个0次项bb来描述。对于某个样本点，不妨记为xx，xx到超平面(w,b)(w,b)上的投影是点x0x0。记xx到该超平面的距离为γγ，xx对应的类标号为yy。

    w∥w∥w∥w∥是超平面的单位法向量，yy决定了法向量的方向。所以有

    x＝x0+yγw∥w∥x＝x0+yγw∥w∥

    因为x0x0在超平面(w,b)(w,b)上，所以有wTx0+b=0wTx0+b=0

    ⇒wT(x−yγw∥w∥)+b=0⇒wT(x−yγw∥w∥)+b=0

    注意到wTw=∥w∥wTw=∥w∥

    ⇒γ=ywTx+b∥w∥⇒γ=ywTx+b∥w∥

    SVM的目标函数

    我们用γiγi来表示超平面到xixi的距离，那么SVM的优化目标的数学表达是这样的

    maxw,b γ=maxw,by(wTx+b)∥w∥maxw,b γ=maxw,by(wTx+b)∥w∥
    s.t.,γi≥γs.t.,γi≥γ

    现在来化简这个目标函数，首先注意到我们要求的(w,b)(w,b)必须能够将训练数据正确的分成两类，即

    wTxi+b≥1如果yi=1wTxi+b≥1如果yi=1
    wTxi+b≤−1如果yi=−1wTxi+b≤−1如果yi=−1

    我们可以把这个约束写成一个等价但更紧凑的形式

    yi(wTxi+b)≥1yi(wTxi+b)≥1

    由此优化目标的目标部分可以简化为

    maxw,by(wTx+b)∥w∥maxw,by(wTx+b)∥w∥
    ⇒maxw,b1∥w∥⇒maxw,b1∥w∥
    ⇒minw,b∥w∥⇒minw,b∥w∥

    而优化目标的约束部分可以简化为

    γi=yi(wTxi+b)∥w∥≥γγi=yi(wTxi+b)∥w∥≥γ
    ⇒yi(wTxi+b)≥γ∥w∥=1⇒yi(wTxi+b)≥γ∥w∥=1
    yi(wTxi+b)≥1yi(wTxi+b)≥1

    简化后的优化目标可以写成

    minw,b12∥w∥2minw,b12∥w∥2
    s.t.,yi(wTxi+b)≥1s.t.,yi(wTxi+b)≥1

    这里添加的1212以及平方是为之后求导数方便，本质上和原来的式子是等价的。

    目标函数的求解

    由于约束条件是线性的，而目标函数是一个二次的形式，因此这是一个典型的凸优化问题，可以使用经典的拉格朗日乘数法来求解。该方法的原理在所有的微积分/数学分析教材上都有，就不再赘述了。

    先写出拉格朗日函数

    Lp=12∥w∥2−∑i=1nλi(yi(wTxi+b)−1)Lp=12∥w∥2−∑i=1nλi(yi(wTxi+b)−1)

    然后对LpLp关于ww和bb求导，令导数值等于00，以最小化拉格朗日函数

    ∂Lp∂w=0⇒w=∑i=1Nλiyixi∂Lp∂w=0⇒w=∑i=1Nλiyixi
    ∂Lp∂b=0⇒∑i=1Nλiyi=0∂Lp∂b=0⇒∑i=1Nλiyi=0

    现在我们有了p+1p+1个方程，但是我们却有2p+12p+1个未知量(pp个拉格朗日乘数λiλi，bb和ww向量中的pp个未知量)。这是因为我们之前的约束条件是不等式的形式而不是等式的形式。

    为此我们可以使用一个叫KKT的技巧，将约束条件变换为

    λi(yi(wTxi+b)−1)=0λi(yi(wTxi+b)−1)=0

    然后将求偏导数之后的式子代入到之前的的拉格朗日函数中，得到拉格朗日函数的对偶形式。

    Lp=∑i=1nλi−12∑i,jλiλjyiyjxixjLp=∑i=1nλi−12∑i,jλiλjyiyjxixj

    只留下拉格朗日乘数和训练数据，然后就可以求解了。

    不过这种解法对于内存的消耗非常大，空间复杂度为O(n2)O(n2)，因此对于大数据集需要使用二次规划来求解。

    R语言有一个kernlab的包提供了SVM的求解，代入我们的数据集可以求得

    #R语言代码
library(kernlab)

model <- ksvm(Species ~ ., data = iris, type = 'C-svc', kernel = 'vanilladot')
plot(model,data = iris)

    图中蓝色越深的地方说明出现在该处的样本点越有可能属于Setosa类，红色越深的地方则反之，右侧的标度表示不同颜色区域离决策平面的距离，可以看到学习的效果是非常好的。
